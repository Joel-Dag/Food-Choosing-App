## 🚀 About  
🔥 Made for a guy who couldn’t choose what to eat 😭  
A smart food selection app that helps you decide instantly.  
No more overthinking — just click and get your next meal 😎  

---

## ✨ Features  
- 🎲 Random food generator  
- 🤖 AI-powered suggestions  
- ⚡ Fast and simple UI  
- 📱 Mobile-friendly  
- 🍽️ Custom food lists  

---

## 🛠️ Tech Stack  
- 💻 Frontend: HTML, CSS, TypeScript  
- ⚙️ Backend: Node.js  
- 🧠 AI Logic: Python (optional)
